# Rad ADRs

## Install

### Linux

```
mkdir /tmp/adr-install
wget -qO- https://github.com/domudall/adr/releases/download/v0.2.0/adr_0.2.0_linux_amd64.tar.gz | tar xvz -C /tmp/adr-install
sudo mv /tmp/adr-install/adr /usr/local/bin/
```
