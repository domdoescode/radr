# Rad ADRs

## Install

### Linux

```
mkdir /tmp/radr-install
wget -qO- https://github.com/domudall/radr/releases/download/v0.3.0/radr_0.3.0_linux_amd64.tar.gz | tar xz -C /tmp/radr-install
sudo mv /tmp/radr-install/radr /usr/local/bin/
```

### Mac

```
mkdir /tmp/radr-install
curl -sSfL https://github.com/domudall/radr/releases/download/v0.3.0/radr_0.3.0_darwin_amd64.tar.gz | tar xz -C /tmp/radr-install
sudo mv /tmp/radr-install/radr /usr/local/bin/
```
