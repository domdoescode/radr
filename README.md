# Rad ADRs

## Install

### Linux

```
mkdir /tmp/radr-install
wget -qO- https://github.com/domudall/radr/releases/download/v0.3.0/adr_0.3.0_linux_amd64.tar.gz | tar xvz -C /tmp/radr-install
sudo mv /tmp/radr-install/radr /usr/local/bin/
```
